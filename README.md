## 🌐 About the Project

**Clearity Care** is a premium landing page for a genomic diagnostics startup based in Amsterdam. The platform is designed to give families who have experienced pregnancy loss access to advanced genetic analysis — providing scientific answers where medicine has historically left a gap.

This landing page serves two audiences:
- **Parents** seeking answers after a loss
- **Investors** interested in backing meaningful healthcare innovation

---

## ✨ Features

- **Animated Hero Section** — Dynamic headline word rotation with a subtle scientific background visual
- **Statistics Section** — Animated counters revealing key data points on pregnancy loss
- **YouTube Explainer Video** — Embedded video explaining the mission
- **How It Works** — 4-step process with custom SVG icons and interactive cards
- **Founders Section** — Real photography of the founding team with credential badges
- **Lead Capture Form** — Dual-tab form for Parents and Investors with full validation
- **Closing CTA** — Emotional closing section with waitlist sign-up
- **Fully Responsive** — Optimised for mobile, tablet, and desktop
- **GDPR Compliant Copy** — Privacy-first messaging throughout
- **SEO Optimised** — Meta descriptions, semantic HTML, and proper heading hierarchy

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/saikrishna406/Clearity_care.git
cd Clearity_care

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
